A machine learning experiment in Java that performs a multivariate linear regression based on 
randomly generated points using a gradient descent algorithm.
Q1 is for one variable while Q2 is multivariate.

There are 5 folders (Q1-1, Q1-2, Q2-1, Q2-2, Q2-3) and this
file(README.txt). Inside each of the folders,there are the corresponding java files(Assignment.java,
LinearRegression.java, StudentInfo.java, and Display.java for Q1 only) and a doc folder the contains
two folders for the java documentation for the Assignment.java and LinearRegression.java in the 
corresponding folder.
